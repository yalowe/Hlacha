# 🧪 מדריך בדיקות - למען שמו באהבה

## 📱 הוראות להתקנה ובדיקה מרחוק

### דרישות מקדימות
- 📱 סמארטפון (Android או iPhone)
- 📶 חיבור לאינטרנט
- ⏱️ כ-10 דקות להתקנה ובדיקה

---

## 🚀 התקנה מהירה (מומלץ)

### אופציה 1: Expo Go (הכי פשוט)

1. **התקנת Expo Go**
   - Android: [הורידו מ-Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent)
   - iPhone: [הורידו מ-App Store](https://apps.apple.com/app/expo-go/id982107779)

2. **פתיחת האפליקציה**
   - פיתחו את Expo Go
   - סרקו את ה-QR code שתקבלו (או שלחו לכם לינק)
   - האפליקציה תיטען תוך 1-2 דקות

3. **אם יש בעיה בטעינה**
   - ודאו שאתם מחוברים לאינטרנט
   - סגרו ופתחו מחדש את Expo Go
   - נסו שוב לסרוק את ה-QR code

## 📋 מה לבדוק?

### 1. ✅ מסך הבית (Dashboard)
- [ ] המסך נטען מהר (תוך 1-2 שניות)
- [ ] רואים "ברוך הבא"
- [ ] יש כפתור "המשך לימוד" (אם למדתם בעבר)
- [ ] יש כפתור "ההלכה היומית"
- [ ] יש כפתור "פרשת השבוע"
- [ ] יש מספר למעלה עם כמות הסימנים (2008)

### 2. 📖 עיון בתוכן
- [ ] לחצו על "עיון" בתפריט התחתון
- [ ] רואים רשימת סימנים
- [ ] לחצו על סימן כלשהו
- [ ] הטקסט מוצג בצורה ברורה
- [ ] אפשר לגלול למעלה ולמטה
- [ ] הטקסט בעברית וקריא

### 3. 🔍 חיפוש
- [ ] לחצו על "חיפוש" למעלה
- [ ] חפשו מילה עברית (לדוגמא: "שבת")
- [ ] התוצאות מופיעות
- [ ] לחיצה על תוצאה פותחת את הסימן הנכון

### 4. ⭐ סימניות
- [ ] לחצו על "סימניות" בתפריט התחתון
- [ ] בחרו סימן ולחצו על כוכב למעלה
- [ ] הסימן נוסף לסימניות
- [ ] חזרו ל"סימניות" - הסימן מופיע שם

### 5. ❓ שאלות ותשובות
- [ ] לחצו על "שאלות" בתפריט התחתון
- [ ] רואים רשימת שאלות
- [ ] אפשר לסנן לפי קטגוריה
- [ ] אפשר לחפש שאלות
- [ ] לחצו על "שאל שאלה" - פותח טופס

### 6. 📊 פרשת השבוע
- [ ] לחצו על "פרשת השבוע" במסך הבית
- [ ] מופיעה פרשת השבוע הנוכחית
- [ ] הטקסט מוצג יפה עם גרדיאנט כחול-סגול
- [ ] אפשר לקרוא את כל הפרשה

### 7. 🔄 ניווט
- [ ] כפתור "חזרה" עובד בכל מסך
- [ ] מעבר בין מסכים מהיר וחלק
- [ ] לא קורסים\נתקעים
